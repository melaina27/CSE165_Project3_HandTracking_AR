# CSE165_Project3_HandTracking_AR
Task 2: Scanned the main passage of CSE 3219 and a passage line of seats, creating a T-shaped map using MRUK in Unity

Task 3: Using ray casting with the right index finger to point to the destination; using the left thumb up gesture to confirm and lock selection, while the left thumb down gesture unlocks selection. When the destination is locked, the right thumb up gesture initiates the agent to walk towards the destination. During travelling, the agent stops once the right fist is up (all fingers bent).
